Delete suppression
~~~~~~~~~~~~~~~~~~

.. code::

    DELETE /suppressions/{suppressionId}

Delete a suppression from your account.

.. note::
   You can delete a suppression at any point, even while it is active.
   If an active suppression is deleted, it ends immediately.
